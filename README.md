## GH-TL
Github Timeline Embedder

<br/>

### How to use?

Just use it as an image tag

```html
<img src="https://gh-tl.now.sh/svg?user=joelibaceta"/>
```

Result

![](https://gh-tl.now.sh/svg?user=joelibaceta)

### Options

**Zoom**

```html
<img src="https://gh-tl.now.sh/svg?user=joelibaceta&zoom=0.5"/>
```
